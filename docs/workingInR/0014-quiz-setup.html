## Checker

Now we can use the checker package to check that your computer has the correct versions of R and RStudio installed, the required packages, and that the recommended RStudio options are set.

First install the `checker` package.

```{r}
#| label: checker-1
#| eval: false
install.packages("checker")
```

Now run

```{r}
#| label: checker-2
#| eval: false
checker::chk_requirements("https://raw.githubusercontent.com/biostats-r/biostats/main/checker/basic.yaml")
```

Please address any issues `checker` finds before continuing to the next chapter.
